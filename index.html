Instructions

1. CD into the directory where you'll be committing.

2. Type "git init".

3. Type "git status" to see where you're at and ensure all your files are listed in the staging area.

4. Type "git add ." to add all of your files.

5. Type "git commit -m 'first commit'" to denote as such.

6. Type "git remote add origin [your github link here]" to begin linking your project to github.

7. Type "git push -u origin master" to push everything you have to github.

8. Type "git status" to ensure you're up to date on master.

9. Repeat steps 4, 5 (changing your commit descriptors), and 7 when you make changes. Repeat step 8 whenever to check where you're at.

FINISHED!!!
